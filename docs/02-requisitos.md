# Requisitos — Site do DA-ADS

## Requisitos Funcionais

Escopo do MVP (fonte da verdade atualizada: board do Trello, coluna Backlog/Ready):

- **Home** — apresentação do DA, chamada para redes sociais.
- **Sobre o DA** — o que é o Diretório Acadêmico, missão e importância.
- **Diretoria/Membros** — membros da gestão atual, com cargo.
- **Contato** — e-mail e formas de contato.
- **Links para redes sociais** — Instagram do DA.
- **Calendário** — eventos do DA + datas acadêmicas oficiais do IFPE.
- **Seção de Membros** — Detalhes dos integrantes do DA.


## Requisitos Não-Funcionais

- **Responsivo / mobile-first** — público entra principalmente pelo celular.
- **Performance** — carregamento rápido, sem dependências pesadas além do
  Bootstrap via CDN.
- **Acessibilidade básica** — contraste adequado, HTML semântico, `alt` em
  imagens.
- **Hospedagem** — GitHub Pages, deploy automático a partir da `main`.
